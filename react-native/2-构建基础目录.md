## 提取文件
### 组件写法
1. ES6
```
export default class HelloComponent extents React.Component{
  render(){
    return <Text style={{fontSize:20,backgroundColor:'red'}}>Hello, {this.props.name}</Text>
  }
}
```
2. ES5
```
var HelloComponent = React.createClass({
  render(){
    return <Text style={{fontSize:20,backgroundColor:'red'}}>Hello, {this.props.name}</Text>
  }
})
module.exports = HelloComponent
```
3. 函数式，无状态不能使用this
```
function HelloComponent(props){
  return <Text style={{fontSize:20,backgroundColor:'red'}}>Hello, {this.props.name}</Text>
}
module.exports = HelloComponent
```
4. 使用
```
import HelloComponent from './HelloComponent'
export default class App extends React.Component{
  render(){
    return (
      <View>
        <HelloComponent/>
      </View>
    )
  }
}
```

### 将app.js文件拆分成独立模块
目录树如下：
```
├── App.js
├── App.test.js
├── README.md
├── app
│   ├── containers
│   │   └── app.js
│   └── index.js
├── app.json
├── package.json
└── yarn.lock
```
1. 修改App.js
```
import React from 'react';
import { AppRegistry } from 'react-native';

import Root from './app/index'

export default Root

AppRegistry.registerComponent('Root', () => Root);
```
2. app作为主要代码文件夹，引出index.js用来装载各种redux、store等库
```
import React from 'react';
import App from './containers/app';

const Root = () => (
  <App />
);

export default Root;
```
3. 初始代码移入 container/app.js中
```
import React from 'react';
import { StyleSheet, Text, View } from 'react-native';
import { Button } from 'antd-mobile';

class App extends React.Component {
  render() {
    return (
      <View style={styles.container}>
        <Text>Open up App.js to start working on your app!</Text>
        <Text>Changes you make will automatically reload.</Text>
        <Text>Shake your phone to open the developer menu.jj</Text>
        <Button>default</Button>
      </View>

    );
  }
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',

  },

```
